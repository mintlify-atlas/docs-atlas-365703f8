librdkafka is the de facto standard C/C++ client library for Apache Kafka. It provides Producer, Consumer, and Admin clients with a battle-tested API used by language bindings across the ecosystem — including Python, Go, .NET, Ruby, and more.

## Key features

<CardGroup cols={2}>
  <Card title="Exactly-Once Semantics" icon="shield-check" href="/guides/transactions-eos">
    Full EOS support with idempotent and transactional producers conforming to KIP-98.
  </Card>
  <Card title="High Performance" icon="bolt" href="/guides/performance-tuning">
    Over 1M msgs/sec producer and 3M msgs/sec consumer throughput on modern hardware.
  </Card>
  <Card title="Security" icon="lock" href="/guides/security">
    SSL/TLS transport and SASL authentication including GSSAPI, SCRAM, and OAUTHBEARER/OIDC.
  </Card>
  <Card title="Statistics" icon="chart-bar" href="/guides/statistics">
    Built-in JSON metrics for producers, consumers, brokers, and partitions.
  </Card>
</CardGroup>

## Get started

<Steps>
  <Step title="Install librdkafka">
    Install via your platform's package manager or build from source. See [Installation](/installation).
  </Step>
  <Step title="Configure your client">
    Create a configuration object and set `bootstrap.servers` along with any security settings. See [Configuration](/concepts/configuration).
  </Step>
  <Step title="Produce or consume messages">
    Use `rd_kafka_produce()` or `rd_kafka_consumer_poll()` to send and receive messages. See the [Quickstart](/quickstart).
  </Step>
  <Step title="Handle delivery reports">
    Call `rd_kafka_poll()` regularly to serve callbacks and check delivery status.
  </Step>
</Steps>
